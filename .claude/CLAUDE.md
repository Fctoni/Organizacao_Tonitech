# Garage Inventory Vault

This Obsidian vault is a garage inventory tracker (one note = one physical item).

For **any** inventory operation — finding where an item is, updating quantity when
the user takes or adds stock, registering a new item, or checking what's running low —
**invoke the `garage-inventory` skill**. It is the single source of truth for the
schema and workflows. Do not reconstruct the rules from memory.
