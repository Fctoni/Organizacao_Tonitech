# AM2302 Temperature & Humidity Sensor

Wired version of the DHT22. Capacitive humidity sensor + thermistor, single-wire digital output (not Dallas 1-Wire compatible — each sensor needs its own data pin). Temperature −40 to 80 °C (±0.5 °C), humidity 0–100 %RH (±2 %RH). Supply 3.3–5.0 V; new reading available roughly every 2 s.
